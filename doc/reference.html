    str2sv [Procedure] (str_pkg.vhdl 79)
    Name and type

     str2sv [Procedure]
    Description

    String to std_logic_vector. This procedure takes a string s and fills the std_logic_vector sv. When the strings are from an input file and you want to report the line where an error was detected use the line_n argument. This procedure aborts execution if the string and std_logic_vector doesn't have the same length or if the it finds a character that isn't 0 or 1.

    pad0r [Function] (str_pkg.vhdl 114)
    Name and type

     pad0r [Function]
    Description

    Adds as many 0s to the right of v to obtain the desired size If the size of the input is bigger than the specified this function returns the MSBs.

    The vectors must be in downto order and use 0 as right bound.
